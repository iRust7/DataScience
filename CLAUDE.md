# Task UTS Data Science: Project Tengah Semester

## Konteks Tugas

Saya memiliki tugas Ujian Tengah Semester untuk mata kuliah **Data Science**.  
Tugas ini berbentuk presentasi hasil praktikum atau project tengah semester.

Project yang dibuat harus berupa **Data Science Model** menggunakan salah satu algoritma berikut:

- Linear Regression
- Logistic Regression

Dataset yang digunakan boleh berupa:

- Dataset numerical
- Dataset categorical
- Dataset dari repository public
- Dataset private

Namun dataset yang dipilih harus memenuhi syarat:

- Masih original
- Tidak terlalu umum
- Belum banyak pembahasan atau tutorial yang sama
- Cocok untuk dibuat model Data Science
- Bisa digunakan untuk task analytical berupa **Estimasi**

---

## Tujuan Utama

Buat sebuah project Data Science lengkap yang mencakup tahapan:

1. Business Understanding
2. Data Understanding
3. Exploratory Data Analysis atau EDA
4. Data Science Model
5. Evaluasi hasil model
6. Kesimpulan akhir

Project ini harus menggunakan pendekatan analisis estimasi dengan algoritma:

- **Linear Regression**, jika target berupa nilai numerik
- **Logistic Regression**, jika target berupa kategori atau klasifikasi

---

## Kriteria Dataset

Carikan dataset yang cocok dengan ketentuan berikut:

- Dataset berasal dari repository public terpercaya atau dataset private
- Dataset masih original dan belum terlalu banyak dibahas
- Tidak memilih dataset yang terlalu umum seperti Titanic, Iris, House Price klasik, atau dataset yang sudah sangat sering dipakai
- Dataset sebaiknya memiliki konteks nyata dan menarik
- Dataset harus punya variabel target yang jelas
- Dataset harus bisa dianalisis dengan Linear Regression atau Logistic Regression
- Dataset harus memungkinkan dilakukan EDA dan preprocessing

Preferensi topik dataset:

- Bencana alam
- Banjir
- Curah hujan
- Polusi udara
- Kualitas udara
- Lingkungan
- Iklim
- Kesehatan publik
- Risiko bencana
- Topik lain yang masih relevan dan tidak terlalu umum
